## 7. Decisions & why
REST not tRPC (simpler backend, manual DTO sync cost acceptable). Radix UI standalone (lighter, full control, no shadcn). axios + TanStack Query (familiar, powerful). Supabase Auth (RLS-safe, SSR-safe, JWT). Biome (single tool, fast, good defaults). Server Components default (Next.js 16 best practice, less JS). CSS custom properties (consistent, themeable, smaller CSS). decimal.js (match backend precision). Light-only theme (professional medical app).
